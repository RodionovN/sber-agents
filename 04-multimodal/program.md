# AI-driven разработка ИИ-агентов

## Цель программы

**Освоить прикладную разработку ИИ-агентов** — от глубокого понимания архитектуры языковых моделей до создания production-ready агентских систем с интеграцией в корпоративные процессы и развертыванием во внешней среде.

## Описание программы

Специализированная образовательная программа для Middle+/Senior разработчиков и аналитиков, направленная на освоение современных технологий создания ИИ-агентов. Программа охватывает полный цикл разработки агентских систем: от фундаментальных принципов работы LLM до практического применения AI-driven методологии, создания RAG-систем, интеграции с внешними инструментами и обеспечения безопасности.

**Целевая аудитория:** Middle+/Senior разработчики, аналитики с навыками программирования

### Ключевые особенности программы:
- **Единый практический проект** — создание ИИ-агента для банковского обслуживания клиентов
- **Итерационный подход** — эволюция проекта от базового чат-бота до автономного агента с RAG
- **80% практики** — каждое занятие включает практическую работу, домашние задания с итерационным развитием проекта
- **AI-driven методология** — использование кодовых ИИ-агентов (Cursor) для ускорения разработки
- **Production-ready решения** — акцент на создании систем, готовых к промышленному использованию (мониторинг, оценка качества, безопасность)
- **Современный технологический стек** — LangChain, LangGraph, MCP, локальные модели

### Практические кейсы из опыта экспертов (Agents, RAG):

- **Корпоративный ИИ-ассистент для сотрудников РЖД** — обработка запросов сотрудников по нормативным документам с персонализацией ответов для повышения осведомленности и информированности о льготах и выплатах
- **ИИ-менеджер по продажам интернет-магазина запчастей** — автоматизация продаж с интеграцией в базу данных и поддержкой 24/7
- **ИИ-аналитик ситуационного центра** — голосовой помощник для анализа данных и отчетов с мультимодальным поиском
- **ИИ-агент службы поддержки по продуктам 1С** — консультирование клиентов по настройке и использованию продуктов на базе 1С с интеграцией в техническую документацию
- **ИИ-ассистент медиа-агентства MediaWise** — мультимодальный поиск в корпоративной базе знаний с поддержкой графиков и таблиц  
- **ИИ-ассистент службы поддержки видеохостинга RUTUBE** — обработка запросов пользователей с семантической классификацией
- **ИИ-ассистент МФЦ СПб** — помощь операторам в работе со сложными и быстро меняющимися регламентами
- **ИИ-ассистент редактора новостного издания** — автоматизация подготовки публикаций через сбор фактологической и визуальной информации по инфоповодам, генерацию альтернативных версий новостных текстов и дополнительных изображений
- **ИИ-ассистент агронома** — мультимодальная обработка отчетов агрономов в чате с автоматическим структурированием данных из текста, голоса и фото в сводную таблицу для оперативного анализа и принятия решений

**Комплексные системы:**
- **Система умного архива** — обработка 3 млн документов с OCR рукописных текстов и построением графа связей
- **Интеллектуальная система карьерных траекторий** — анализ 1 млн вакансий для формирования образовательных траекторий
- **Система интеллектуального поиска видео для видеохостинга RUTUBE** — семантический поиск с предобработкой запросов и персонализацией контента

## Формат обучения

**12 занятий по 1,5 часа** с сочетанием теории (15%) и практики (85%). К каждому занятию предусмотрены домашние задания для самостоятельной отработки навыков.

## Программа курса

### Тема 1. Основы LLM и ИИ-агентов

Вводная тема курса, посвященная фундаментальным принципам работы языковых моделей и основам создания ИИ-агентов. Вы познакомитесь с архитектурой современных LLM, их возможностями и ограничениями, а также получите представление о том, как эти знания применяются при разработке агентских систем.

**Программа:**
- Архитектура и принципы работы языковых моделей
- Ограничения и возможности современных LLM
- Методы работы с моделями: промпт-инжиниринг, контекст-инжиниринг, RAG
- Что такое агент: определение, компоненты, архитектура
- Обзор моделей и провайдеров
- Обзор дорожной карты практики курса

---

### Тема 2. Разработка ИИ-ассистента с LangChain

Практическое погружение в разработку ИИ-ассистентов с использованием фреймворка LangChain. Вы научитесь работать с OpenAI API и совместимыми интерфейсами, создавать структурированный вывод, управлять поведением ассистента через системные промпты и создадите свой первый базовый ИИ-ассистент.

**Программа:**
- OpenAI API и совместимые интерфейсы для работы с моделями
- Структурированный вывод (Structured Output)
- Основы LangChain для создания ИИ-приложений
- Работа с различными провайдерами и смена моделей
- Системные промпты и управление поведением ИИ-ассистента
- Создание базового ИИ-ассистента

**Практика:** Создание базового ИИ-ассистента для банковского обслуживания клиентов.

---

### Тема 3. AI-driven разработка с ИИ-агентами на примере Cursor

Освоение современной методологии AI-driven разработки с использованием кодовых ИИ-агентов. Вы научитесь эффективно использовать Cursor для ускорения разработки, управлять контекстом, применять продвинутые техники промптинга и организовывать рабочий процесс от идеи до развертывания.

**Программа:**
- AI-driven методология и её применение для эффективной разработки с использованием ИИ-агентов
- Настройка Cursor: конфигурация, правила, соглашения, рабочий процесс
- Управление контекстом и эффективные техники промптинга
- От идеи к реализации: планирование итерации, соглашения, списки задач, рабочий процесс
- Разработка по всему жизненному циклу (от идеи до развертывания)

**Практика:** Применение AI-driven методологии для разработки и развития проекта ИИ-ассистента.

---

### Тема 4. Мультимодальные возможности и локальный запуск LLM

Расширение возможностей ИИ-ассистента за счет мультимодальности: работа с голосом, изображениями и различными форматами данных. Также вы научитесь запускать языковые модели локально для обеспечения конфиденциальности и снижения зависимости от облачных сервисов.

**Программа:**
- Обработка голосовых сообщений и транскрибация речи
- Анализ изображений и визуальная экспертиза
- Задачи извлечения информации из различных форматов
- Локальный запуск моделей
- Prompt engineering
- In-context learning

**Практика:** Добавление мультимодальных возможностей в ИИ-ассистента: обработка голоса и изображений.

---

### Тема 5. Технология RAG: паттерны и архитектура

Глубокое погружение в технологию Retrieval-Augmented Generation (RAG) — ключевой подход для создания ИИ-систем, работающих с большими базами знаний. Вы изучите различные архитектурные паттерны, стратегии chunking, векторные базы данных и эволюцию подходов от Naive RAG до Agentic RAG.

**Программа:**
- Обзор RAG-экосистемы: классификация подходов и архитектур
- RAG pipeline: индексация, chunking, векторизация, поиск, генерация
- Стратегии chunking: размер, перекрытие, семантическое разделение
- Векторные базы данных и embedding модели
- Naive RAG vs Advanced RAG: эволюция подходов
- Self-RAG и Agentic RAG: системы с самооценкой и планированием
- Кейсы применения RAG

---

### Тема 6. Создание RAG-ассистента с LangChain

Практическое занятие по созданию полнофункционального RAG-ассистента с использованием LangChain. Вы пройдете через все этапы RAG pipeline: от загрузки и индексирования документов до создания диалогового ассистента с контекстом из базы знаний.

**Программа:**
- Настройка окружения и загрузка документов
- Индексирование: загрузчики документов, разбиение на чанки, создание векторного хранилища
- Retrieval: поиск релевантных фрагментов документов
- Augmented Generation: создание промптов с контекстом и генерация ответов
- Создание RAG-цепочек для автоматизации процесса
- Диалоговый режим и обработка истории сообщений

**Практика:** Создание RAG-ассистента для банковского обслуживания с базой знаний о продуктах и услугах.

---

### Тема 7. Мониторинг и оценка качества

Критически важная тема для создания production-ready систем. Вы научитесь оценивать качество работы ИИ-агентов с помощью метрик, автоматической оценки и специализированных инструментов мониторинга. Освоите подходы к отладке и комплексному мониторингу агентских систем.

**Программа:**
- Базовые метрики оценки качества: precision, recall, faithfulness, relevance
- LLM-as-a-Judge: автоматическая оценка ответов агентов
- RAGAS и другие фреймворки автоматической оценки
- LangSmith: мониторинг и оценка качества работы ИИ-агентов
- Способы отладки: как понимать взаимодействие функций внутри агента
- Реализация комплексного мониторинга работы ассистента

**Практика:** Внедрение мониторинга и автоматической оценки качества в RAG-ассистента.

---

### Тема 8. Агенты: теория и паттерны построения

Теоретическая основа для создания сложных агентских систем. Вы изучите различия между агентами и workflow, познакомитесь с архитектурными паттернами, принципами построения мультиагентных систем и лучшими практиками их применения.

**Программа:**
- Agent vs Workflow: что и когда применяется
- Агентные архитектурные паттерны
- Мультиагентные системы: координация, коммуникация, специализация
- Сценарии применения агентных систем
- Инструменты для создания агентов: архитектура и интеграция
- Лучшие практики применения агентных паттернов

---

### Тема 9. LangGraph: создание сложных ИИ-агентов

Практическое освоение LangGraph — мощного фреймворка для создания сложных агентских workflow. Вы научитесь проектировать и реализовывать автономных агентов с циклами принятия решений, планированием и обработкой ошибок, используя ReAct паттерн.

**Программа:**
- LangGraph: графовые workflows для сложных агентов
- Состояния, узлы, ребра: построение агентского workflow
- Создание Agentic RAG-системы с планированием запросов
- Реализация автономного агента для банковского обслуживания клиентов
- ReAct паттерн: Reasoning + Acting в агентском контексте
- Отладка и трейсинг агентских процессов
- Циклы принятия решений и обработка ошибок

**Практика:** Создание автономного агента с LangGraph для комплексного банковского обслуживания.

---

### Тема 10. Agents + MCP: расширение возможностей через инструменты

Изучение Model Context Protocol (MCP) — современного стандарта для расширения возможностей LLM через интеграцию с внешними системами и инструментами. Вы научитесь создавать собственные MCP-серверы и интегрировать ИИ-агенты с корпоративными системами.

**Программа:**
- Model Context Protocol (MCP): стандарт для расширения возможностей LLM
- Архитектура MCP
- Создание MCP-серверов: ресурсы, инструменты, промпты
- Интеграция ИИ-агентов с внешними системами через MCP
- Разработка собственных MCP-серверов

**Практика:** Разработка MCP-сервера для интеграции агента с банковскими системами.

---

### Тема 11. Безопасность и оценка качества агентных систем

Критически важная тема для промышленного внедрения агентов. Вы изучите методы многоуровневого тестирования агентских систем, метрики качества, подходы к оценке процесса мышления агента, а также уязвимости и защитные механизмы для обеспечения безопасности.

**Программа:**
- Многоуровневое тестирование агентов
- Метрики качества: task success, correctness, reliability, efficiency, safety
- Process Reward Models: оценка процесса мышления, а не только результатов
- Стандартизированные бенчмарки и тестовые наборы для агентских систем
- Уязвимости агентов: prompt injection, jailbreaking, data poisoning
- Защитные механизмы и фреймворки безопасности

**Практика:** Внедрение комплексной системы безопасности и тестирования в агентскую систему.

---

Завершающее занятие курса, посвященное презентации и разбору итоговых проектов участников. Обсуждение архитектурных решений, выбора технологий, проблем и найденных решений. Ответы на накопившиеся вопросы и рекомендации по дальнейшему развитию и внедрению в production.

**Программа:**
- Презентация итоговых проектов участников
- Демонстрация созданных ИИ-агентов
- Разбор архитектурных решений и технических подходов
- Анализ выбора стека технологий и моделей
- Обсуждение проблем и найденных решений в процессе разработки
- Ответы на накопившиеся вопросы по темам курса
- Разбор реальных кейсов внедрения ИИ-агентов
- Рекомендации по дальнейшему развитию и внедрению в production
